# PrepTrack

PrepTrack is a modern placement preparation dashboard designed for students preparing for technical interviews and campus placements.

## Features
- DSA Progress Tracker
- Daily Goal Management
- Company Preparation Dashboard
- Dark/Light Mode
- Responsive UI
- LocalStorage Support
- Search & Filter Functionality

## Tech Stack
- HTML5
- CSS3
- JavaScript
